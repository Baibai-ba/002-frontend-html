关于本站

这里是一个个人练习与展示的前端小站，主要目标是：

  • 用最基础的 HTML / CSS / JS 搭建功能。
  • 尝试 组件化拆分，保持目录清晰。
  • 在 GitHub Pages 上托管，方便随时预览。

当前功能

  • 👋 签到：点击按钮，日历会记录今日签到。
  • 🎮 猜数字游戏：经典 1~100 猜数，10 次机会。
  • 📱 自适应布局：在手机/电脑上都能正常访问。
  • 🔗 导航/404 页面：统一风格，方便跳转。

后续计划

  1. ✨ 增加更多小游戏（比如反应力测试、记忆卡片）。
  2. 🏆 添加 排行榜/本地成绩保存。
  3. 🎨 优化主题和样式，让界面更统一。
  4. 📄 增加文档入口（如 git-cheatsheet）。
